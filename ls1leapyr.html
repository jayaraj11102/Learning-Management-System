  3. print("Leap Year")

Sample output:

Enter Year:2020

Leap Year

In the above code, we have taken the user input as an integer using the input() function. Then, we have performed to check a leap year or not based upon the user given year and print the result using the print() statement.

Assume variable year holds 2020
