ARITHMETIC OPERATION

Aim :

To write the Java program to show the various use of the VARIOUS OPERATORS

  1. class Main {
  2. public static void main(String[] args) {
  3. int a = 12, b = 5;
  4. System.out.println("a + b = " + (a + b));
  5. System.out.println("a - b = " + (a - b));
  6. System.out.println("a * b = " + (a * b));
  7. System.out.println("a / b = " + (a / b));
  8. System.out.println("a % b = " + (a % b));
  9. }
  10. }

Sample output:

a + b = 17

a - b = 7

a * b = 60

a / b = 2

a % b = 2

EXPLANATION :

In the above code, we have taken the input as an integer as compile time . Then, we have performed the arithmetical operation on the given numbers and print the result using the System.out.println() statement.

Assume variable a holds 12 and variable b holds 5

OPERATORS

  • + ADDITION :
  • Adds values on either side of the operator.
  • EXAMPLE: a + b = 30
  • - SUBTRACTION :
  • Subtracts right hand operand from left hand operand
  • EXAMPLE: a - b = -10
  • / Division :
  • Divides left hand operand by right hand operand
  • EXAMPLE: b / a = 2
  • * Multiplication :
  • Multiplies values on either side of the operator
  • EXAMPLE: a * b = 200
  • % Modulus :
  • Divides left hand operand by right hand operand and returns remainder
  • EXAMPLE: b % a = 0
  • SYSTEM.OUT.PRINTLN FUNCTION :
  • using System.out.println() method to print the result to the output screen
