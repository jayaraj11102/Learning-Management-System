Aim :

To write the C program to show the grade of student based upon the marks of the student

  1. #include
  2. int main()
  3. {
  4. int english, chemistry, computers, physics, maths;
  5. float Total, Percentage;
  6. printf(" Please Enter the Five Subjects Marks : \n");
  7. scanf("%d%d%d%d%d", &english, &chemistry, &computers, &physics, &maths);
  8. Total = english + chemistry + computers + physics + maths;
  9. Percentage = (Total / 500) * 100;
  10. printf(" Total Marks = %.2f\n", Total);
  11. printf(" Marks Percentage = %.2f", Percentage);
  12. if(Percentage >= 90)
  13. {
  14. printf("\n Grade A");
  15. }
  16. else if(Percentage >= 80)
  17. {
  18. printf("\n Grade B");
  19. }
  20. else if(Percentage >= 70)
  21. {
  22. printf("\n Grade C");
  23. }
  24. else if(Percentage >= 60)
  25. {
  26. printf("\n Grade D");
  27. }
  28. else if(Percentage >= 40)
  29. {
  30. printf("\n Grade E");
  31. }
  32. else
  33. {
  34. printf("\n Fail");
  35. }
  36. return 0;
  37. }

Sample output:

Please Enter the Five Subjects Marks :

70 60 90 80 85

Total Marks = 385.00

Marks Percentage = 77.0

Grade C

EXPLANATION :

In the above code, we have taken the user input as an integer using the scanf() function. Then, we have performed to check a leap year or not based upon the user given year and print the result using the printf() statement.

Assume variable english chemistry computer physics maths holds 70 60 90 80 85 respectively.
