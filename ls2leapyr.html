2) CHECK LEAP YEAR OR NOT

Aim :

To write the C program to check whether it is leap year or not

  1. #include
  2. #include
  3. void main() {
  4. int year;
  5. printf("Enter a year: ");
  6. scanf("%d", &year);
  7. if(((year%4==0) && ((year%400==0) || (year%100!==0))
  8. {
  9. printf("%d is a leap year", &year);
  10. } else {
  11. printf("%d is not a leap year", &year);
  12. }
  13. getch();
  14. }

Sample output:

Enter a Year:2020

2020 is a Leap Year

EXPLANATION :

In the above code, we have taken the user input as an integer using the scanf() function. Then, we have performed to check a leap year or not based upon the user given year and print the result using the printf() statement.

Assume variable year holds 2020
